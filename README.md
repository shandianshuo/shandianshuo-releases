## 关于闪电说

闪电说是一款本地沟通 Agent。桌面端运行在你的电脑上，不只是 AI 语音输入法，还能结合当前屏幕、个人记忆和知识库理解语境，帮助你组织回复、修改表达，并通过技能先完成任务再生成结果。

目前已提供 macOS、Windows 和 Linux 桌面版本，Android 与 iOS 移动版本由合作伙伴独立开发和发布。桌面端可用于微信、企业微信、钉钉、飞书、WhatsApp、浏览器、邮箱、终端等软件，也可以在其他输入框中使用。

#### Linux 安装包怎么选

- **AppImage** — 便携版本，下载后添加执行权限即可运行
- **DEB** — 适用于 Debian、Ubuntu 等使用 DEB 包的发行版
- **RPM** — 适用于 Fedora、RHEL 等使用 RPM 包的发行版

AppImage 首次运行前需要添加执行权限：

```bash
chmod +x shandianshuo_*_amd64.AppImage
./shandianshuo_*_amd64.AppImage
```

Linux 版本目前处于内测阶段。如遇到问题，欢迎通过 [用户反馈](https://shandianshuo.featurebase.app) 告诉我们。

## 功能特性

- **直接说** — 短按快捷键开始说话，自动整理结构、去掉重复和口语赘词、修正错字，并将干净文字写入当前输入框
- **帮我说** — 长按快捷键，让 AI 结合屏幕上下文和个人记忆，把你的意图组织成清楚、自然、得体的回复
- **全面回忆** — 持续更新联系人相关的工作偏好、表达习惯、关键事项和历史上下文，让沟通更连贯
- **知识库** — 管理风格、词典、常用回复和产品资料，回复时自动参考，让人名、术语和表达更准确
- **技能** — 执行代码、请求数据或调用工具，先完成任务，再把结果整理成可以直接使用的回复
- **隐私优先** — 支持完全本地的语音识别和自接模型；知识库、记忆、词典与配置保存在本机，处理路径由你选择
